# Sugar-Spice---Kitchen-Dreams
Sugar & Spice: Kitchen Dreams is a zero-dependency, highly responsive procedural cooking and baking simulation game engineered using ES6+, HTML5, and CSS3. The game tasks players with managing active customer orders by preparing diverse recipes across interactive Whisking, Baking, and Icing stations while managing ticking customer patience meters and a shift timer. On the technical side, the project features real-time procedural sound effects synthesized mathematically using the Web Audio API, smooth CSS3 animations for interactive state cues, and a custom Difficulty Manager utilizing exponential scaling formulas to dynamically accelerate customer spawns and decrease burning thresholds as the player’s score increases.
